# Circademic - מערכת ניהול אקדמי חכמה

![Circademic Logo](LOGO.png)

## 🎯 על הפרוייקט

**Circademic** היא מערכת ניהול אקדמי מקיפה המיועדת לסטודנטים במוסדות להשכלה גבוהה. המערכת מאפשרת לסטודנטים לעקוב אחר ציונים, לנהל קורסים, לחשב ממוצעים, ולתכנן את ההתקדמות האקדמית שלהם.

## ✨ תכונות עיקריות

### 📊 ניהול ציונים וקורסים
- **חישוב ממוצע משוקלל** - חישוב אוטומטי מדויק של הממוצע המשוקלל
- **מעקב אחר קורסים** - ניהול מפורט של כל הקורסים עם פרטים מלאים
- **סינון וחיפוש** - חיפוש קורסים לפי שם, סמסטר, קטגוריה
- **מיון דינמי** - מיון לפי שם, ציון, נ"ז, סמסטר

### 📈 ויזואליזציות וגרפים
- **גרף מגמת ציונים** - מעקב אחר ההתקדמות לאורך סמסטרים
- **גרף התפלגות מועדים** - ניתוח מועדי א' ומועדי ב'
- **סטטיסטיקות לפי קטגוריות** - ממוצעים נפרדים לחובה, בחירה, כללי וספורט
- **פס התקדמות** - מעקב אחוז התקדמות בתואר

### 🔮 מחשבונים מתקדמים
- **מחשבון "מה אם"** - סימולציה של ציונים עתידיים והשפעתם על הממוצע
- **מחשבון ציון מגן** - חישוב הציון הנדרש בבחינה כדי לעבור או להגיע למטרה

### 📄 דוחות וייצוא
- **דוח PDF מקצועי** - ייצוא דוח מפורט למלגות או עבודה
- **ייצוא CSV** - שמירת נתונים בפורמט CSV
- **ייבוא CSV** - העלאת קורסים מקובץ CSV

### ☁️ ענן ואבטחה
- **שמירה בענן** - כל הנתונים מסונכרנים דרך Firebase
- **התחברות מאובטחת** - אימות עם אימייל/סיסמה או חשבון Google
- **נתונים מוצפנים** - אבטחה מלאה של כל המידע האישי

### 🎨 עיצוב וחוויית משתמש
- **מצב כהה** - מעבר חלק בין מצב בהיר לכהה
- **רספונסיבי מלא** - מותאם למובייל, טאבלט ושולחן עבודה
- **ממשק אינטואיטיבי** - עיצוב נקי ונוח לשימוש

## 🚀 התקנה והרצה

### דרישות מקדימות
- דפדפן מודרני (Chrome, Firefox, Safari, Edge)
- חיבור לאינטרנט (לשימוש ב-Firebase)

### הוראות התקנה

1. **הורדת הקבצים**
   ```bash
   # שכפל את כל הקבצים לתיקייה מקומית
   ```

2. **הגדרת Firebase**
   - הפרוייקט כבר מוגדר עם Firebase
   - הפרטים נמצאים ב-`firebase-config.js`
   - במידת הצורך, ניתן להחליף לפרוייקט Firebase משלך

3. **הרצה מקומית**
   ```bash
   # הפעל שרת HTTP מקומי
   python -m http.server 8000
   # או
   npx serve
   ```

4. **גישה למערכת**
   - פתח דפדפן וגש ל-`http://localhost:8000`
   - הירשם או התחבר למערכת
   - התחל להוסיף קורסים ולעקוב אחר ההתקדמות!

## 📱 שימוש במערכת

### התחברות ראשונית
1. לחץ על "התחבר" בדף הנחיתה
2. בחר "הירשם" אם אין לך חשבון
3. מלא פרטים או התחבר עם Google
4. תועבר אוטומטית לדאשבורד

### הוספת קורסים
1. בדאשבורד, לחץ על "קורסים" בתפריט
2. לחץ על "הוסף קורס"
3. מלא את כל הפרטים:
   - שם הקורס
   - מספר נ"ז
   - ציון
   - סמסטר
   - קטגוריה (חובה/בחירה/כללי/ספורט)
   - מועד (א'/ב')
   - סוג ציון (רגיל/בינארי)
4. שמור - הקורס יתווסף והנתונים יתעדכנו אוטומטית

### שימוש במחשבונים
#### מחשבון "מה אם"
1. עבור ל"מחשבונים" בתפריט
2. הוסף קורסים עתידיים עם ציונים משוערים
3. בחר קורסים קיימים לשיפור (סימולציה של מועד ב')
4. לחץ "חשב" לראות את הממוצע החזוי

#### מחשבון ציון מגן
1. הזן ציון מגן ואחוז המגן
2. (אופציונלי) הזן ציון בחינה - לקבלת הציון הסופי
3. ללא ציון בחינה - המחשבון יחשב כמה צריך בבחינה כדי לעבור

### ייצוא דוחות
1. עבור ל"דוחות" בתפריט
2. בחר "דוח PDF מקצועי" לדוח מלא
3. או "ייצוא CSV" לקובץ נתונים גולמי
